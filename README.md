# jQuery.MovingState
jQuery.MovingStateは、連続で発動されるイベントの、「開始時」「稼動時」「停止時」のタイミングが取れるjQueryのプラグインです。
scrollイベント、resizeイベント、mousemoveイベントなどで有用です。
